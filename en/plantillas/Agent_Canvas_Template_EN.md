# AGENT CANVAS — Editable Template

**Design Your AI Agent** — Framework adapted from the Business Model Canvas

---

## 📋 HOW TO USE THIS CANVAS

1. **Start with Value Proposition and User Segment** — don't try to fill in all blocks at once
2. **Validate each block with real users** before moving to the next
3. **Review and update** the canvas every 2–3 weeks during development
4. **This is not a final document** — it's a thinking and design tool
5. **Blocks are numbered** in suggested working order (1→9)

## 🎯 SUGGESTED WORKING ORDER

1. **User Segment** → Who is it for?
2. **Value Proposition** → What problem does it solve?
3. **Channels** → How does it reach the user?
4. **Relationship** → What tone and personality?
5. **KPIs** → How do we measure success?
6. **Resources** → What knowledge does it need?
7. **Activities** → What skills does it have?
8. **Partners** → What systems does it integrate with?
9. **Costs/Risks** → What does it cost? What risks exist?

---

## 💡 PRACTICAL TIPS

- ✅ **DO:** Use the canvas collaboratively with your team
- ✅ **DO:** Validate each block with real users
- ✅ **DO:** Iterate and update the canvas every 2–3 weeks
- ✅ **DO:** Use sticky notes or collaborative tools (Miro, Mural...)

- ❌ **DON'T:** Try to fill in the entire canvas in a single session
- ❌ **DON'T:** Treat the canvas as a final, immutable document
- ❌ **DON'T:** Design the agent alone — involve users from day 1
- ❌ **DON'T:** Move forward if the Value Proposition is not clear
