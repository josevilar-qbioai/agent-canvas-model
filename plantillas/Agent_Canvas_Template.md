# AGENT CANVAS — Plantilla Editable

**Diseña tu Agente IA** — Framework adaptado del Business Model Canvas

---

## 📋 CÓMO USAR ESTE CANVAS

1. **Empieza por Propuesta de Valor y Segmento** — no intentes rellenar todos los bloques a la vez
2. **Valida cada bloque con usuarios reales** antes de pasar al siguiente
3. **Revisa y actualiza** el canvas cada 2-3 semanas durante el desarrollo
4. **No es un documento final** — es una herramienta de pensamiento y diseño
5. **Los bloques están numerados** en orden sugerido de trabajo (1→9)

## 🎯 ORDEN SUGERIDO DE TRABAJO

1. **Segmento** → ¿Para quién?
2. **Propuesta de Valor** → ¿Qué problema resuelve?
3. **Canales** → ¿Cómo llega al usuario?
4. **Relación** → ¿Qué tono y personalidad?
5. **KPIs** → ¿Cómo medimos éxito?
6. **Recursos** → ¿Qué conocimiento necesita?
7. **Actividades** → ¿Qué skills tiene?
8. **Socios** → ¿Qué sistemas integra?
9. **Costes/Riesgos** → ¿Qué cuesta? ¿Qué riesgos?

---

## 💡 CONSEJOS PRÁCTICOS

- ✅ **SÍ:** Usa el canvas colaborativamente con tu equipo
- ✅ **SÍ:** Valida cada bloque con usuarios reales
- ✅ **SÍ:** Itera y actualiza el canvas cada 2-3 semanas
- ✅ **SÍ:** Usa post-its físicos o herramientas colaborativas (Miro, Mural...)

- ❌ **NO:** Intentes rellenar todo el canvas en una sola sesión
- ❌ **NO:** Trates el canvas como un documento final e inmutable
- ❌ **NO:** Diseñes el agente solo — involucra a usuarios desde el día 1
- ❌ **NO:** Sigas adelante si la Propuesta de Valor no está clara
